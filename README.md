# lterm
lterm is a small script built to install a bash hook for full terminal logging.

## Install lterm

Install via git

```
git clone https://github.com/killswitch-GUI/zlib_wrapper.git
python setup.py install
```

Install via pip

```
pip install lterm
```

## Run lterm
open up a terminal:
```
lterm.py -h
```
